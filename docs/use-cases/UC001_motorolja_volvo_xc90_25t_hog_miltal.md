## 📋 Sammanfattning

Detta use case analyserar vilket motoroljeVal som är optimalt för en Volvo XC90 2.5T
(motor B5254T2) med **350 000 km** på mätaren. Motorn har överskridit sin normala
livslängd (~300 000 km) och kräver särskild hänsyn vid val av olja avseende viskositet,
tätningsegenskaper och slitskydd.

## 🔍 Analys: Varför Högre Viskositet vid 350 000 km?

### ⚙️ Motorns Tillstånd vid Högt Miltal

B5254T2-motorn har en förväntad livslängd på cirka **300 000 km**. Vid 350 000 km
förväntas följande slitage:

- **Cylinderväggarna** har ökat spel mot kolvringarna
- **Ventilstyrningar** har slitits, ger ökad oljepassage
- **Vevlager och ramlager** har marginellt ökade toleranser
- **Kamaxeltätningar** kan läcka eller vara hårda
- **Turbolagring** har ökat spel

### 🌡️ Viskositetslogik

| Faktor                        | Lägre viskositet (0W-30)                  | Högre viskositet (5W-40 / 10W-40)            |
| ----------------------------- | ----------------------------------------- | -------------------------------------------- |
| **Oljefilm vid slitna lager** | Tunnare film, risk för metallkontakt      | Tjockare film, bättre skydd                  |
| **Oljeförbrukning**           | Högre (passerar lättare förbi kolvringar) | Lägre (tjockare olja stannar bättre)         |
| **Kallstart**                 | Bättre flöde vid kallstart                | Något trögare, men 5W klarar nordiskt klimat |
| **Tätningseffekt**            | Sämre vid slitna tätningar                | Bättre — "fyller ut" slitna toleranser       |
| **Turbo-smörjning**           | Snabbare initiell smörjning               | Bättre skydd vid drifttemperatur             |
| **Bränsleförbrukning**        | Marginellt lägre                          | Marginellt högre                             |

### ✅ Rekommendation: Gå Upp till 5W-40

Vid 350 000 km rekommenderas att gå från fabrikens **0W-30** till **5W-40** av
följande skäl:

1. **Tjockare oljefilm** vid drifttemperatur skyddar slitna lager bättre
2. **Minskad oljeförbrukning** — tjockare olja passerar inte lika lätt förbi kolvringar
3. **5W** ger fortfarande utmärkt kallstartsegenskaper i svenskt klimat (ner till -30°C)
4. **ACEA A3/B4** uppfylls av alla rekommenderade produkter
5. **Turbo-anpassat** — 5W-40 rekommenderas specifikt för turbomotorer

> ⚠️ **OBS:** Gå _inte_ till 10W-40 i Norden om bilen används vintertid.
> 10W-klassen ger sämre kallstartssmörjning under -15°C. Välj 5W-40 som kompromiss.

## 🔧 Viktiga Underhållsåtgärder vid 350 000 km

### ⚠️ PCV-system (Vevhusventilation)

B5254T2 är **känd** för problem med PCV-systemet (Positive Crankcase Ventilation)
vid högt miltal. Symptom:

- Ökad oljeförbrukning
- Rök från oljepåfyllningslocket
- Ojämn tomgång
- Oljeläckage vid tätningar (tryckuppbyggnad i vevhuset)

**Åtgärd:** Byt oljefälla (flame trap), tillhörande slangar och packningar.
Rekommenderas vid 100 000–120 000 km — om det inte är gjort vid 350 000 km
bör det prioriteras **före** oljebyte.

> 💡 **Diagnostik:** Sätt en gummihandske eller ballong på oljemätstickans rör.
> Om den blåser upp snabbt = högt vevhustryck = PCV behöver service.

### 🔄 Rekommenderat Bytesintervall vid 350 000 km

| Parameter            | Fabriksrekommendation  | Rekommendation vid 350 000 km |
| -------------------- | ---------------------- | ----------------------------- |
| **Bytesintervall**   | 30 000 km / 12 månader | **10 000 km / 6 månader**     |
| **Kontrollera nivå** | Varannan tankning      | **Vid varje tankning**        |
| **Oljefilterbyte**   | Vid varje oljebyte     | Vid varje oljebyte            |
| **PCV-service**      | 100 000–120 000 km     | **Kontrollera omedelbart**    |

### 📝 Kompletterande Rekommendationer

1. **Använd INTE oljetillsatser** — Volvo avråder uttryckligen från detta
2. **Motorspolning** (engine flush) kan göras var tredje oljebyte med t.ex. Liqui Moly
   Engine Flush — hjälper rensa slam i äldre motorer
3. **Kontrollera turbo-oljematning** — slangarna kan bli spröda vid högt miltal
4. **Byt kamaxeltätningar** om det syns oljeläckage vid ventilkåpan
5. **Kontrollera oljetrycket** — om varningslampan blinkar vid tomgång,
   överväg tjockare olja (10W-40) eller undersök oljetrycksgivaren
